Final Project Overview

by Andrew V.

PROJECT OVERVIEW: Objective

  • The goal is to enable people to enjoy music with others around them.
  • It does so by aggregating everyones' playlist in the same physical space and playing back songs that the majority would enjoy.

PROJECT OVERVIEW: Physical Description

  • Web Application built with Spotify Web API
    • Users (on Spotify) would access the Web App and authorize the app to pull his/her playlist from Spotify
    • The Web App would aggregate all playlists in the area and create a Group playlist based on shared attributes
  • Visualization
    • Informational graphics of the current track and playlist visualized with Processing
